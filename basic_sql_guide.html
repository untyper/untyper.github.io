SQL (Structured Query Language) is used to manage and manipulate relational databases. Below is a simple guide covering the basics.

1. SQL Database Structure

A relational database consists of:

  • Tables (Store data in rows and columns)
  • Columns (Define data types like text, numbers, dates)
  • Rows (Records) (Each row is a single entry)
  • Primary Keys (Unique identifier for each row)
  • Foreign Keys (Links between tables)

2. Creating a Database

CREATE DATABASE my_database;
USE my_database;

3. Creating a Table

CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(50),
    email VARCHAR(100) UNIQUE
  );

4. Inserting Data into a Table

INSERT INTO users (name, email) VALUES ('Alice', 'alice@email.com');
INSERT INTO users (name, email) VALUES ('Bob', 'bob@email.com');

5. Reading Data from a Table (SELECT)

SELECT * FROM users;
SELECT name, email FROM users;
SELECT * FROM users WHERE name = 'Alice';

6. Updating Data

UPDATE users SET email = 'alice123@email.com' WHERE name = 'Alice';

7. Deleting Data

DELETE FROM users WHERE name = 'Bob';
DELETE FROM users;

8. Joining Tables

SELECT users.name, orders.product, orders.price 
FROM users
JOIN orders ON users.id = orders.user_id;

9. Other Useful SQL Commands

Sorting Results

SELECT * FROM users ORDER BY name ASC;

Counting Rows

SELECT COUNT(*) FROM users;
